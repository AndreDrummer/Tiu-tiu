Tiu, Tiu is a powerful and intuitive app created to facilitate the adoption and donation process for animals. üöÄ Technologies Used

Tiu, Tiu was built with a set of modern technologies to offer the best user experience:

  • Flutter: Framework used to ensure a smooth app on both Android and iOS.
  • Firebase: Implemented for authentication, real-time database, and push notifications.
  • REST API: For efficient and fast data synchronization.
  • MVVM Architecture: To keep the code organized and ensure maintainability and scalability.
